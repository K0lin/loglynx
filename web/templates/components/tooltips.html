{{/* Common tooltip definitions - can be called from any page */}} {{/* Overview Page Tooltips */}} {{define "tooltip-total-requests"}}
Total Requests
Total number of HTTP requests received in the selected time period. Includes all status codes and request types.
 {{end}} {{define "tooltip-success-rate"}}
Success Rate
Percentage of successful requests (2xx and 3xx status codes). Higher is better - aim for 95%+.
 {{end}} {{define "tooltip-avg-response"}}
Response Time
Average time taken to process and respond to requests. Lower values indicate better performance.
 {{end}} {{define "tooltip-unique-visitors"}}
Unique Visitors
Number of unique IP addresses that have accessed your service. Used to estimate distinct users.
 {{end}} {{define "tooltip-error-rate"}}
Error Rate
Percentage of failed requests (4xx client errors and 5xx server errors). Lower is better.
 {{end}} {{define "tooltip-bandwidth"}}
Bandwidth Usage
Total amount of data transferred to clients. Useful for monitoring data usage and costs.
 {{end}} {{/* Performance Page Tooltips */}} {{define "tooltip-p50"}}
P50 Percentile
Median response time - 50% of requests are faster than this value. Good baseline metric.
 {{end}} {{define "tooltip-p95"}}
P95 Percentile
95th percentile - 95% of requests are faster than this. Helps identify outliers.
 {{end}} {{define "tooltip-p99"}}
P99 Percentile
99th percentile - 99% of requests are faster than this. Important for user experience.
 {{end}} {{/* Traffic Page Tooltips */}} {{define "tooltip-peak-traffic"}}
Peak Traffic
Maximum number of requests per hour during the selected period. Useful for capacity planning.
 {{end}} {{define "tooltip-countries"}}
Geographic Reach
Number of different countries accessing your service. Indicates global reach.
 {{end}} {{define "tooltip-heatmap"}}
Traffic Heatmap
Visual representation of traffic patterns by day and hour. Darker colors indicate higher traffic.
 {{end}} {{/* Geographic Page Tooltips */}} {{define "tooltip-geo-coordinates"}}
Coordinates
Latitude and longitude of the IP address location. Approximate location based on GeoIP data.
 {{end}} {{define "tooltip-map-clusters"}}
Map Clustering
Groups nearby markers for better map visualization. Click to zoom in and see individual locations.
 {{end}} {{/* Security Page Tooltips */}} {{define "tooltip-status-codes"}}
Status Codes
HTTP status codes indicate request outcomes. 2xx=success, 3xx=redirect, 4xx=client error, 5xx=server error.
 {{end}} {{define "tooltip-protocol"}}
Protocol Version
HTTP protocol version used for requests. HTTP/2 and HTTP/3 offer better performance.
 {{end}} {{define "tooltip-tls-version"}}
TLS Version
TLS/SSL version used for encrypted connections. TLS 1.2+ is recommended for security.
 {{end}} {{/* Real-time Page Tooltips */}} {{define "tooltip-live-metrics"}}
Live Metrics
Real-time data streaming every 2 seconds. Shows current system activity.
 {{end}} {{define "tooltip-request-rate"}}
Request Rate
Current rate of incoming requests per second. Calculated from last minute of data.
 {{end}} {{/* Users Page Tooltips */}} {{define "tooltip-browser-distribution"}}
Browser Stats
Shows which web browsers your visitors use. Helps optimize for popular browsers.
 {{end}} {{define "tooltip-os-distribution"}}
OS Distribution
Operating system breakdown of visitors. Useful for platform-specific optimization.
 {{end}} {{define "tooltip-bot-detection"}}
Bot Detection
Automated detection of bot traffic vs human visitors. Bots include search engines, scrapers, etc.
 {{end}} {{/* Content Page Tooltips */}} {{define "tooltip-top-paths"}}
Popular Paths
Most frequently accessed URLs on your site. Helps identify popular content.
 {{end}} {{define "tooltip-http-methods"}}
HTTP Methods
Distribution of HTTP request methods (GET for reading, POST for submitting, etc.).
 {{end}} {{/* Backends Page Tooltips */}} {{define "tooltip-backend-health"}}
Health Status
Health status based on error rate and response time. Green=healthy, Yellow=degraded, Red=critical.
 {{end}} {{define "tooltip-backend-name"}}
Backend Service
Name of the backend service handling requests. Configured in your reverse proxy.
 {{end}} {{/* General Tooltips */}} {{define "tooltip-time-range"}}
Time Range
Select time period to analyze. Longer periods show trends, shorter periods show recent activity.
 {{end}} {{define "tooltip-auto-refresh"}}
Auto Refresh
Automatically reload data at the specified interval. Disable for detailed analysis.
 {{end}} {{define "tooltip-service-filter"}}

 {{end}} {{define "tooltip-export-data"}}
